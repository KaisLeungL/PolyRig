# Sources

All web evidence is GitHub official documentation (and one IETF standard);
cross-pack architecture rules cite local PolyRig pack contracts and this
repository's registry spec as user-provided evidence. Volatile endpoint paths,
scope names, and package versions are routed to `deps.yaml`, not pinned in
prose.
